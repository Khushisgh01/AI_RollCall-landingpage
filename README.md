## Local Setup
### Prerequisites
Node.js installed if the project uses a build tool
Or a simple browser environment if it is a static site
### Steps
Clone the repository:
```bash
git clone https://github.com/Khushisgh01/AI_RollCall-landingpage.git
cd AI_RollCall-landingpage
```
Open the project locally:
If it is a static HTML/CSS/JS site, open index.html in your browser.
If it uses a local dev server, install dependencies first:
```bash
npm install
npm run dev
```
Visit the local URL shown by your dev server, or open the HTML file directly.
